Introducing Randomness
  • +Reference: random()
  • +random() returns a floating point number
  • +random() without params returns a value between 0 and 1 (e.g., 0.5108338863145219)
  • +random(someNum) returns a value between 0 and someNum
  • +random(someNum) returns a value between 0 and someNum

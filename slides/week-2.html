Introducing Randomness
  • +Reference: random()
  • +random() returns a floating point number
  • +random() without params returns a value between 0 and 1 (e.g., 0.5108338863145219)
  • +random(someNum) returns a value between 0 and someNum
  • +random(min, max) returns a value between min (inclusive) and max (exclusive)

Repeating Forms: Using For Loops
So we don't have to repeat ourselves a lot!
  • +Instead of writing ellipse(xPos, yPos, d, d) ten times,
  • +We can use something called a for loop
  • +Notice that we are starting at 0, which is very common in programming 
  •                   
                    // We can break this first line into three parts:

                    // let i = 0 means we start our loop at 0
                    // i < 10 means we will stop looping when we reach 9
                    // i++ means that every time we loop, we will increment i by 1

                    for (let i = 0; i < 10; i++) {
                      print(i);
                      // 0
                      // 1
                      // 2
                      // ...
                      // 9
                    }
Repeating Forms: By Hand
How do we draw 10 circles? 
            
              function setup() {
                createCanvas(400, 400);
                noStroke();
                frameRate(10);
              }
              
              function draw() {
                background(240);
                fill(100, 140, 255);
                
                let d = 30; // circle diameter
                let startX = 40; // x coord for our first circle

                for (let i = 0; i < 10; i++) {
                  ellipse(startX, 200, d, d);

                  // we want to update the x position for each circle drawn
                  startX += 35;
                }
              }

Constraining Bounds: If statements
How can we keep our circle in view? 
            
              var xPos; // starting x position
              var yPos; // starting y position
              
              function setup() {
                createCanvas(400, 400);
              
                // assign our x and y positions
                x = 200;
                y = 200;
              }
              
              function draw() {
                background(0);
                fill(0, 125, 255);
              
                // Draw a circle
                stroke(0, 255, 255);
                strokeWeight(2);
                ellipse(x, y, 40, 40);
              
                // Moving up at a constant speed
                y = y - 5;
                
                // Here we check if the circle has moved offscreen
                if (y <= 0) {
                  y = 400;
                }
              }

Setting up a P5 Sketch
createCanvas() & background()
  • +Functions sometimes have 'parameters' or 'arguments'
  • +Parameters are specified after a function's name, inside of parentheses, for example:
  • +createCanvas(400, 400) is a function that sets up a 'canvas' to hold your art
  • +400, 400 are parameters specifying the dimensions of your canvas
  • +background(0, 0, 0) is a function that sets the background of your canvas
  • +0, 0, 0 are 3 parameters that specify a color, in this case, black

Drawing Basic Shapes
The Coordinate System
  • +(0, 0) starts at the top left corner of your canvas
Drawing Basic Shapes
ellipse(xPosition, yPosition, width, height)
  • +You should see a white circle, white is the default fill color
  • +We draw a circle at (150, 150), with a width and height of 100px 
  •                   
                    function setup() {
                      createCanvas(400, 400);
                    } 
                    
                    function draw() { 
                      background(0, 0, 0);
                      ellipse(150, 150, 100, 100);
                    }
Drawing Basic Shapes
rect(topLeftX, topLeftY, bottomRightX, bottomRightY)
  • +You should see a white rectangle, white is the default fill color
  • +We draw a rectangle using the top left corner and bottom right corner coordinates
  • +top left corner at (50, 50), bottom right corner at (250, 100) 
  •                   
                    function setup() {
                      createCanvas(400, 400);
                    } 
                    
                    function draw() { 
                      background(0, 0, 0);
                      rect(50, 50, 250, 100);
                    }
Color: Fill & Stroke
fill(redValue, greenValue, blueValue)
  • +Colors are defined with 3 values, corresponding to red, green, and blue color channels
  • +Each color channel can have a value between 0-255
  • +(0, 0, 0) will be black, and (255, 255, 255) will be white
  • +To change the fill color of the circle, we have to set it before drawing the ellipse
  • +Play around with color combinations! 
  •                   
                    function setup() {
                      createCanvas(400, 400);
                    } 
                    
                    function draw() { 
                      background(0, 0, 0);
                      fill(130, 40, 150);
                      ellipse(150, 150, 100, 100);
                    }
Color: Fill & Stroke
stroke(redValue, greenValue, blueValue) & strokeWeight(amount)
  • +Your shapes can have borders, or a 'stroke'
  • +stroke(0, 255, 255) sets the color of the stroke
  • +strokeWeight(5) sets the thickness of the stroke (in pixels) 
  •                   
                    function setup() { 
                      createCanvas(400, 400);
                    } 
                    
                    function draw() { 
                      background(0, 0,0);
                      fill(130, 40, 150);
                      strokeWeight(5);
                      stroke(0, 255, 255);
                      ellipse(150, 150, 100, 100);
                    }
